C) How to add your data in JSON DATABASE - top

Screenshot of JSON DATABASE:


1- You need firstly to create an empty file with name db.json in anywhere on your PC
2- Then open this file with any editor
3- Copy and past the below code inside of your file

[
{
"wallpaper_index":"1",
"wallpaper_name":"put_the_NAME_of_your_WALLPAPER_here",
"wallpaper_site_name":"put_the_NAME_of_the_wallpaper_SITE_here",
"wallpaper_site_url":"put_the_LINK_of_the_wallpaper_SITE_here",
"wallpaper_url":"put_the_LINK_of_the_WALLPAPER_here"
}
]

4- follow the this tutorial to know how to add or edit any image in JSON DATABASE, download the tutorial here:Tutorial Video
5-finally when you finished your database, upload it to your server then copy it's link then past it in the value of wallpaperDataBase at SettingsClasse.java file

D) HoW TO change Admob Units Ids - top




1 - Go to SettingsClasse.java
2 - Replace admBanner and Interstitial with your Admob Id's Units.
Replace contactMail value with you Email
Replace privacy_policy_url value with you Privacy policy link
Replace more_apps_link value with you more apps link


Replace wallpaperDataBase value with your database link
